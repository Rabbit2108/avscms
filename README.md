## 什么是 AVSCMS?

AVSCMS！是一个内容管理系统 (CMS)，可让您构建网站和强大的视频共享应用程序。 它是一个简单而强大的 Web 服务器应用程序，需要具有 PHP 和 MySQL 的服务器才能运行。

您可以找到完整的技术要求 [here](https://www.avscms.com/product/features/).

AVSCMS! 现在是根据 GNU 通用公共许可证 v3.0 或更高版本分发的免费开源软件。
